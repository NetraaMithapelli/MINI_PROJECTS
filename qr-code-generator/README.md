#QR Code Generator (Python)

This mini project generates QR codes from text or URLs using Python.
The program converts a given link into a scannable QR image that can be saved locally.

## Technologies Used
* Python
* qrcode library
* Pillow (PIL) for image processing

## Features
* Generate QR code from any URL or text
* Save the QR code as a PNG image
* Customize QR code size, border, and colors
* High error correction for better scanning

## Example
Example input:
https://www.karanaujlamusic.com/
Output:
A downloadable QR code image that opens the website when scanned.

## Installation
Install the required libraries:
pip install qrcode[pil]

## Run the Program
python qr_code.py
